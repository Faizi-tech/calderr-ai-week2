# Day 1 – Advanced Prompt Engineering

## Overview

Day 1 focused on learning the fundamentals of prompt engineering and implementing a Chain of Thought (CoT) prompting pipeline using LangChain and the Groq API.

## Objectives

- Understand Prompt Engineering
- Learn Chain of Thought (CoT)
- Learn Tree of Thoughts (ToT)
- Learn Self Consistency
- Practice prompt rewriting
- Build a CoT prompting pipeline
- Compare Normal Prompting and CoT Prompting

## Topics Covered

- Prompt Engineering
- Chain of Thought (CoT)
- Tree of Thoughts (ToT)
- Self Consistency
- Prompt Design Best Practices
- LangChain
- Groq API
- Llama 3.3 70B Versatile

## Tasks Completed

- Studied Prompt Engineering concepts.
- Explained CoT, ToT, and Self Consistency.
- Rewrote 10 poorly written prompts into well-structured prompts.
- Configured the Python environment and virtual environment.
- Connected to the Groq API using environment variables.
- Initialized a LangChain `ChatGroq` model.
- Built reusable Normal Prompt and Chain of Thought prompting pipelines.
- Compared Normal Prompting and CoT Prompting on 10 reasoning problems.
- Recorded observations and conclusions.
